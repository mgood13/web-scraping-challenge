# web-scraping-challenge
## This program is intended to scrape images from the JPL and NASA websites and display them on the webpage. There are 5 main scraping tasks accomplished by this program.


#### Scrape Initialization: This is the title of the webpage and has the button which the user can press in order to scrape the desired data.
  <p><img width="638" alt="Page_Title" src="https://user-images.githubusercontent.com/35506304/101975642-0d93db80-3c0c-11eb-8a68-b45662997e2b.png"></p>
  
#### Featured Article Scrape: This scrapes the NASA news page and returns information from the first article.

<p><img width="741" alt="Featured_Article" src="https://user-images.githubusercontent.com/35506304/101975712-809d5200-3c0c-11eb-95d9-62fba64eeb90.png"></p>

#### Featured Image Scrape: This function scrapes from the image database. Then it takes the featured image and displays its title and image.
#### Mars Data Scrape: This scrapes the data table from the Mars Facts page and displays them in a table format.

<p> <img width="1077" alt="Featured_Image Table" src="https://user-images.githubusercontent.com/35506304/101975730-a9bde280-3c0c-11eb-8d9d-b6ac3fa4558f.png"></p>
